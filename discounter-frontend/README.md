# discounter-frontend

## Zweck

Vue-Frontend fuer PLZ-, Regions- und Haendlerabfragen gegen das lokale Backend.

## Voraussetzungen

- Node.js 20+
- npm 10+
- laufendes Backend unter `http://localhost:8080`

## Installation

```bash
npm --prefix discounter-frontend install
```

## Start

```bash
npm --prefix discounter-frontend run dev
```

Frontend-URL:

```text
http://localhost:5173
```

## Checks

```bash
npm --prefix discounter-frontend run test
npm --prefix discounter-frontend run build
```
