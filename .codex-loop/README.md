# Codex Loop

Dieser Ordner ist bewusst versioniert.

Er zeigt den letzten bekannten Laufzustand des lokalen Autopilot-Loops:

- `issue.md`: aktuell bearbeitetes Issue
- `worker.md`: letzte Worker-Ausgabe
- `review.md`: letzte Reviewer-Ausgabe
- `doc-worker.md`: letzte Doku-Worker-Ausgabe
- `state.json`: kompakter Laufstatus

Änderungen in diesem Ordner blockieren den Loop-Start nicht und werden nicht in Issue-Commits aufgenommen.
